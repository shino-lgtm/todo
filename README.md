これは毎日のタスクを楽しく管理できる、かわいいToDoリストです🎀  
パステルピンクで統一し、ローカル保存でデータも安心♪

🌸 [実際のサイトはこちら](https://shino-lgtm.github.io/todo/)

- ローカルストレージに保存されるから安心✨
